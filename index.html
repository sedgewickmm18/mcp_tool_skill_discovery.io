Dynamic Skill Discovery Tools

Advanced discovery using Granite embeddings and cosine similarity

Discovery Tools:

search_skills(query: str)
Uses Granite embeddings with cosine similarity to find matching skills based on natural language queries. Returns ranked list of best matches with similarity scores.
load_skill(skill_id: str)
Loads the full skill definition into LLM context with "_meta": {"uiVisibility": "hidden"}. Skill becomes available for immediate use without UI clutter.

💡 Skills remain hidden from UI but accessible when needed via intelligent discovery
